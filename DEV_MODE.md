# 开发模式配置

## 快速调试时间设置

为了方便调试通知和状态切换功能，项目支持开发模式时间配置。

### 启用开发模式

1. 复制 `.env.example` 为 `.env`：
   ```bash
   cp .env.example .env
   ```

2. 编辑 `.env` 文件，取消注释相应行：
   ```env
   # 开发模式时间设置（秒）
   DEV_WORK_TIME=25        # 工作时间：25秒
   DEV_BREAK_TIME=5        # 短休息：5秒  
   DEV_LONG_BREAK_TIME=15  # 长休息：15秒
   LONG_BREAK_INTERVAL=4   # 每4个番茄钟后长休息
   ```

3. 重新启动应用：
   ```bash
   npm run dev
   ```

### 时间对照

| 模式 | 工作时间 | 短休息 | 长休息 |
|------|----------|---------|--------|
| 生产模式 | 25分钟 | 5分钟 | 15分钟 |
| 开发模式 | 25秒 | 5秒 | 15秒 |

### 注意事项

- `.env` 文件已添加到 `.gitignore`，不会提交到版本控制
- 删除 `.env` 文件或注释掉配置即可恢复生产模式
- 开发模式下可以快速测试通知、状态切换等功能
